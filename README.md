# ImageProcessing

Python image processing project with a GUI.

## Requirements

- Python 3.10+

## Quick Start (macOS / zsh)

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

## If `.venv` Already Exists

```bash
source .venv/bin/activate
python main.py
```

## Deactivate Virtual Environment

```bash
deactivate
```
